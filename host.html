Sala de Consulta Virtual

Bienvenido al espacio de videollamada. Como anfitrión, podrá gestionar la consulta y los participantes.

Instrucciones para el anfitrión

  1. Asegúrese de tener una conexión estable a internet.
  2. Utilice auriculares para mejorar la calidad del audio.
  3. Puede silenciar a los participantes si es necesario.
  4. Comparta su pantalla para mostrar documentos relevantes.
  5. Finalice la llamada cuando la consulta haya terminado.
